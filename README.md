# Iris OS

Iris is a high-performance web-based desktop environment built with Next.js, Zustand, and Tailwind CSS.

## Features
- **Window Layout Manager**: High-performance layout orchestration with virtual workspace containment, drag snapping grids, and keyboard shortcut mapping.
- **Virtual File System (VFS)**: Client-side storage and node tree serialization.
- **System Utilities**: Real-time Terminal shell (xterm.js), dynamic settings, and quick command execution overlays.
- **Gmail Client**: Premium 3-pane email client with unread highlights.
- **Telegram Client**: Secure MTProto chat simulator featuring ghost message recovery and edit trace capture.
